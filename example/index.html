HappySlider

Examples:

  • Basic example
  • Initialize with data attribute
  • Swipe slider for slide effect
  • Auto height

Usage

HappySlider requires jQuery and Modernizr to run. Download latest version of jQuery and Modernizr and place them to your page.

Add HappySlider to your page.

<link rel="stylesheet" href="happyslider/happyslider.css"/>
<script src="happyslider/jquery.happyslider.min.js"></script>

Initialization happyslider plugin

        <script>
$(function() {
    $(".js__happyslider").happyslider();
});   
</script>

Or initialize with data attribute

<div class="happyslider" data-happyslider data-happyslider-options='{"height": 420, "width": 640, "animateType": "scale"}'>
    ...
</div>

Options:

  • width
  • height
  • infiniti
  • animateType
  • animateDuration
  • animateEase
  • animateCallback

Roadmap:
